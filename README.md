## 如何搭建开发环境？

* 新建一个Algorithms-4th的Java项目
* 下载[相关jar包](https://github.com/Crearns/Algorithms-4th/blob/master/algs4.jar?raw=true)以及测试数据
* 将下载的jar包导入至项目中
